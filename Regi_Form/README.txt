Title:
Registration Form using Visual Studio

Framework used:
This project is created using Visual Studio 2010 IDE.

Steps for creating project in visual studio.
1)open visual studio 2010.
2)select file -> New -> website -> select ASP.NET Empty website ->give name to website ->click OK.
3)write code for .aspx file

Description about Project
This project displays simple registration form.
In which user enters their details and when it click on submit button user's details is displayed in one label and if user wants to remove their details they can by clicking on the Remove button.
